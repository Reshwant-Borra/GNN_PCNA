# Overview

GNN-PCNA Phase 2 is a governed rebuild. The primary workflow is Codex using `wiki/` as persistent memory and `docs/scientific_governance/` as binding scientific law.

## Current Project Shape

- `AGENTS.md` tells Codex how to work.
- `wiki/index.md` is the memory/navigation dashboard.
- `docs/scientific_governance/` defines rules, gates, and constraints.
- `crawls/` contains source leads, raw metadata, and prior context, not automatic truth.
- V1 and old artifacts are historical reference only.

## Working Objective

Build a fresh, provenance-controlled Phase 2 pipeline for residue-level PCNA candidate cryptic/allosteric/pocket-like region prediction and auditing, without repeating V1 leakage, provenance, MD, or claim-control failures.

## Hard Boundaries

- No training before readiness and human review gates.
- No invented scientific assumptions.
- No reuse of stale V1 artifacts without audit.
- No clinical, therapeutic, drug-discovery, or mechanism validation claims.
