## Git Ignore Rules

The following should NOT be committed to git (add to `.gitignore`):

```
data/raw/*.pdb
data/processed/
data/graphs/
data/labels/
data/trajectories/
```

Use git-lfs if you need to version large files.
Document what was acquired and how in `DATA_PROVENANCE.md`.
