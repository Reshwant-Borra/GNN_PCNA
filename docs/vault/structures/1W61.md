# 1W61 — EXCLUDED: NOT PCNA

> **WARNING**: 1W61 is **proline racemase** (*Trypanosoma cruzi*), not PCNA.
> It was incorrectly included in training data and all evaluation scripts.
> All AUROC values computed with 1W61 are invalid. This structure has been removed from
> `KNOWN_PCNA_IDS`, `PCNA_IDS`, and `_PCNA_IDS` in all scripts.

## Why AUROC was 1.0000

Proline racemase has a completely different fold. Any classifier trivially separates it from
PCNA by backbone geometry alone — this is label leakage, not learned pocket recognition.
